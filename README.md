# among use

AMONG USというゲームでユーザーが気軽にイベントを企画したり参加したりするための参加型プラットフォームサービスです。
スクール内の輪を同ゲームを通じて広げていきたいという思いのもと、5人のメンバーで締切駆動のアジャイル開発しています。
